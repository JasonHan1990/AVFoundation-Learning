# AVPlayer Test App

This app contains the core functions of a video player controller.
  - Play a local video
  - Play a video online
  - Jump to selected time
  - Add observers and handle notifications
  - - Playback end
  - - Interruption
  - - Fail to play to end
  - - Route changed
  - - Play did finished
  - - Observing current time
